2.  **Data Regression:** The modified canvas math broke the rendering loop, causing all data (Controllers, Notes, Velocity) to disappear.

**Objective:** You must revert all canvas math regressions to restore the continuous data stream. Then, you will implement the Y-axis labels using standard React/Tailwind DOM elements placed outside and to the left of the canvas container.

## 2. Technical Decisions & Dependencies
* **Reversion:** The canvas `requestAnimationFrame` loop MUST return to using standard `w` and `h` calculations for the `x` coordinates. All instances of `axisWidth` and `plotW` must be deleted.
* **DOM Layout:** You will wrap the canvas container in a `flex-row`. The Y-axis labels will be a `flex-col` with `justify-between`. 
* **Alignment Math:** The canvas uses `padding = h * 0.05` for top and bottom. To perfectly align the HTML numbers with the canvas grid lines, the HTML label container will use `py-[5%]` to match the canvas padding.
